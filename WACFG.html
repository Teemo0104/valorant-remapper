压枪触发（鼠标侧键）

勾选启用，选鼠标按键和触发方式。每把枪占用一个 (按键 × 触发方式) 组合； 多把枪冲突时以列表上方先出现者为准。
触发方式说明：
  • 单击 / 双击：按一次 / 350 ms 内按两次。
  • Ctrl + 单击 / 双击：在按住 Ctrl 时单击 / 双击。
  • 按住 + 滚轮↑ / 滚轮↓：按住所选鼠标键不放，再向上 / 向下滚动一格滚轮即可装弹。 滚轮 不需要 按下去。装弹用滚轮、不影响游戏内滚轮原本的操作（仍然会发送一次正常滚动事件）。
  • B3 中键 + 滚轮↑ / 滚轮↓：选 B3 + 滚轮方向 = "按下滚轮往上/往下滚"。 手感上等于按下滚轮然后立刻滚一格，常用作"无侧键也能压枪"的备选触发。
